Atributos comuns dos botões

  • type
    • submit
    • reset
    • button
  • autofocus
  • disable
  • name
  • value
  • form

Exemplos:

  1. reset button e input text com value = "valor inicial"

    Toda vez que clicar ele reseta para o valor inicial

  2. autofocus

    Caso apertemos "Enter" ele direciona para o botão, por exmeplo

  3. button com name e value

    Quando clicamos ele enviará o valor com o respectivo nome, como se fosse por exemplo:
    button name="button" value="valor" = button: valor

  4. O form como propriedade do botão

    Serve para linkar o botão ao formulário, quando o botão não está dentro do formulário. Veja o exemplo no html abaixo:
