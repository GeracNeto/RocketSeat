Estudo de Posicionamento

Existem 4 tipos de Posicionamento

  1. static
    • Os elementos seguem o fluxo normal
    • Um abaixo do outro
  2. relative
    • Possui as propriedades: top, right, bottom, left, z-index
    • Essas podem ser utilizadas para mover o elemento na tela
    • O espaço inicial do obejto se mantém e pode ser reutlizado para outro coisa
  3. absolute
    • Também possui as 4 propriedades: top, right, bottom, left, z-index
    • O espaço inicial do objeto é automaticamente preenchido
    • Dessa forma é possível sobrepor elementos
    • O absolute sempre fica referente em relação a página e não ao objeto em que está inserido, a não ser que, o elemento pai tenha um position relative
  4. fixed
    • Também possui as 4 propriedades: top, right, bottom, left, z-index
    • O elemento fica fixo na página. Dessa forma, se move com o scroll do mouse
